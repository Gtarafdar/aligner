Media Manager

Extract and download media assets from websites

Show Media Manager panel to scan and download images, SVGs, videos, fonts, icons, and Lottie files

Automatically scan the page for media assets when Media Manager is activated

What Media Manager Can Find:

  • Images: <img> tags, background images, srcset, picture elements
  • SVG: Inline SVG, external SVG files, SVG backgrounds
  • Videos: <video> tags, source elements, embedded videos (YouTube, Vimeo)
  • Fonts: @font-face declarations (WOFF, WOFF2, TTF, OTF)
  • Icons: Favicons, apple-touch-icon, manifest icons, og:image
  • Lottie: JSON animation files, Lottie player elements

💡 Usage Tips:

  • Click "Scan Page" to detect all media on the current page
  • Use tabs to filter by media type
  • Preview, copy URL, or download individual items
  • Use "Download All" to batch download all items of a type
  • Drag the panel to reposition it on the page
  • Minimize the panel to a floating button when not in use

Color Picker

Pick colors from any element on the page and manage color palettes

Show Color Picker panel to pick colors from any webpage element

Default format for copied color values

Maximum number of colors to keep in history (10-100)

Automatically open Color Picker panel when feature is enabled

How to Use Color Picker:

  1. Click the "✎ Color Picker" button in the toolbar or popup
  2. Click "🎯 Pick Color from Page" in the panel
  3. Hover over any element to preview its color
  4. Click the element to pick and save the color
  5. Click any format (HEX/RGB/HSL) to copy to clipboard
  6. Save color collections as named palettes

What you can pick: Background colors, text colors, border colors, gradient colors, and more!

How to Use Palette Generator:

  1. Enable the feature and click the Palette button in the popup
  2. Click "Scan Page" in the panel to extract colors
  3. View the generated palette organized by similarity
  4. Copy individual colors by clicking on swatches
  5. Export entire palette as CSS variables, JSON, or plain text
  6. Save palettes for later reference with custom names
