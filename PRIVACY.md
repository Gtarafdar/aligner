# Aligner privacy

Aligner is a Chrome extension that runs in your browser.

## What stays local

- Feature settings and workspace preferences are stored with Chrome’s extension storage on your device.
- Page tools (rulers, inspect, media scan, etc.) operate on the tab you enable them on.
- Optional WordPress helper features talk only to the WordPress site you are using, when you are logged in and choose those tools.

## What we do not do

- No account system
- No analytics or telemetry endpoint shipped in this repository
- No selling of browsing data

## Optional helper plugin

The bundled **Aligner WP Tools Helper** is an optional WordPress plugin you install yourself. It exposes nonce-protected endpoints for Role switcher and private Theme preview for authorized users on that site.
