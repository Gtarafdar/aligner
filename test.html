Aligner Extension Test Page

This page is designed to test the Aligner Chrome extension. Use the keyboard shortcuts or popup to activate features:

  • Ctrl+Shift+R - Toggle Rulers
  • Ctrl+Shift+G - Toggle Guides
  • Ctrl+Shift+L - Toggle Grids (Column/Baseline/Modular)
  • Ctrl+Shift+M - Toggle Measurement Tool

Test the grid system by toggling between different grid types in the options page:

  • Column Grid - Vertical columns with configurable count, gutters, and margins
  • Baseline Grid - Horizontal lines for vertical rhythm
  • Modular Grid - Combination of both column and baseline grids

Rulers

Horizontal and vertical rulers with tick marks and measurements. Adjust color, opacity, and units in settings.

Guides

Create draggable guide lines. Lock them in place or enable snap-to-pixel for precision alignment.

Grids

Column, baseline, or modular grids. Configure columns (1-24), spacing, gutters, margins, and appearance.

Measurement

Point-to-point measurement tool. Click two points to measure distance. Shows width, height, and diagonal.

Toolbar

Floating toolbar for quick feature toggling. Now draggable! Click and drag to reposition anywhere on the page.

Settings

Comprehensive options page accessible from the popup. Customize colors, sizes, opacity, and behavior for all features.

Testing Tips

1. Open the extension popup and toggle individual features on/off

2. Right-click the extension icon and select "Options" to access full settings

3. Try adjusting grid columns (1-24) and see the layout update in real-time

4. Drag the toolbar to different positions on the page

5. Use keyboard shortcuts for quick feature toggling
