Install in under a minute

Chrome Web Store listing may come later. Until then, load the open-source ZIP like any developer extension.

Download the package

Grab aligner.zip from this page or GitHub Releases.

Unzip somewhere permanent

Do not delete the folder after install — Chrome loads the extension from that path.

Load unpacked

Open chrome://extensions, enable Developer mode, click Load unpacked, and select the unzipped folder that contains manifest.json.

Open Aligner Home

Pin the icon, open a page, then use the popup footer Home link — or Settings → Workspace → Open Aligner Home.

Stay up to date

Chrome only auto-updates extensions from the Chrome Web Store. For the GitHub ZIP build: download the latest release, replace your unpacked folder contents, then click Reload on chrome://extensions.

FAQ

Straight answers for install, privacy, and WordPress.

Is Aligner free?

Yes. Aligner is free and open source under the MIT License.

Does it upload my browsing data?

No analytics or account system ships in this project. Settings stay in Chrome storage on your device. See PRIVACY.md.

How do I update after installing from ZIP?

Download the newest aligner.zip from Releases, replace the files in your unpacked folder, then hit Reload in chrome://extensions. Watch the GitHub repo for release notifications. Store auto-update will apply once a Chrome Web Store listing exists.

What is the WordPress helper plugin?

Optional. Most WP Tools work with normal admin access. Role switcher and private Theme switcher need Aligner WP Tools Helper (ZIP bundled in the extension and Releases). Install via Plugins → Add New → Upload.

Will it slow my browser?

Tools run on the tab where you enable them. Use Workspace presets to hide groups you do not need so the popup stays light.
