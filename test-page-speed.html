How to Test

Testing Instructions

  1. Open the extension popup (click the Aligner icon in your toolbar)
  2. Click the "Page Speed" button with icon ≉
  3. The Page Speed sidebar should open on the right side
  4. Click "Run Performance Audit" button
  5. Wait for the audit to complete (should take 1-2 seconds)
  6. Review the performance metrics, Core Web Vitals, and score
  7. Switch between Metrics, Opportunities, and Diagnostics tabs
  8. Click "Show Fix Plan" or "Show Details" buttons to expand accordion sections
  9. Test the draggable panel (drag from the header)
  10. Test minimize button (should create a floating ≉ button)
  11. Test close button (should close the panel)

✅ Expected Results

  • Sidebar opens without errors
  • Panel is draggable from the header
  • Audit completes and shows performance score (0-100)
  • Displays Core Web Vitals: FCP, LCP, TBT, CLS, Speed Index
  • Shows color-coded scores: Green (90+), Orange (50-89), Red (<50)
  • Opportunities tab shows actionable recommendations
  • Diagnostics tab shows detailed analysis
  • Accordion sections expand/collapse properly
  • Minimize creates floating button at bottom
  • Close button removes panel completely
  • No console errors
  • All tabs switch correctly
