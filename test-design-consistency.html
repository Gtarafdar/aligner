Testing Instructions

  1. Load Extension: Ensure Aligner extension is installed and active
  2. Open Panel: Click extension icon → "Design Check" button (Ӫ icon)
  3. Configure Settings:
    • Go to Settings tab
    • Adjust tolerance levels (try strict vs. lenient)
    • Return to Controls tab
  4. Run Scan: Check all three boxes (Typography, Colors, Spacing) and click "Scan for Inconsistencies"
  5. Review Results: Check the Results tab for detected issues
  6. Test UI:
    • Drag panel around the screen
    • Minimize and restore using the "−" button
    • Close panel using the "×" button
    • Re-open from extension popup
  7. Test Tab Switching: Click between Controls, Results, and Settings tabs
  8. Adjust Tolerance: Change tolerance values and re-scan to see different results

Expected Results

Typography Issues:

  • H1 elements use 2-3 different font sizes (36px, 38px)
  • H2 elements use 2 different sizes (28px, 30px)
  • H3 elements use 2 different sizes (22px, 24px)
  • P elements use 3 different sizes (14px, 16px, 18px)
  • Multiple font weights for same elements

Color Issues:

  • Similar blues: #2563eb vs #2564ec vs #2665ed
  • Similar greens: #10b981 vs #11ba82
  • Similar grays: #1f2937 vs #212a38, #6b7280 vs #6c7381
  • Similar light backgrounds: #f9fafb vs #fafbfc
  • Using 9+ different text colors (high count warning)

Spacing Issues:

  • Padding: 10px, 12px, 16px, 18px, 20px, 22px, 24px, 32px
  • Margins: 6px, 8px, 10px, 12px, 16px, 18px, 24px
  • Too many unique spacing values (should be using spacing scale)
