🖼️ Image Replacement Test

Test the image upload and URL replacement features

📋 How to Test:
  1. Press Ctrl+Shift+I (or Cmd+Shift+I on Mac) to activate inspector
  2. Click on any image below
  3. Open the Image & Media Controls tab
  4. Try each replacement method
✅ Test Scenarios:
  1. Upload Local Image: Click "📤 Upload Image" → Select file from computer → Image should change immediately
  2. Replace with URL: Click "🔗 From URL" → Enter URL like https://picsum.photos/400/300 → Image should update
  3. Preview Shows: After replacement, preview section should display the new image
  4. Info Updates: Source URL and dimensions should update
