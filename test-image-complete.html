✨ Expected Results

  1. Upload: File dialog opens, image is replaced immediately after selection
  2. Replace URL: Prompt opens, image updates with entered URL
  3. Remove: IMG elements removed from page, background images cleared
  4. Crop: Prompt for crop values, clip-path applied
