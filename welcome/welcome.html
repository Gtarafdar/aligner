Welcome dashboard

Your visual toolkit, sized to the work you do today.

Aligner packs layout, inspect, color, quality, utility, and WordPress tools into one extension. Most people do not need every toggle every day — so start by choosing a workspace, then learn what each group is for.

Popup, settings, and this home page

The browser popup is for daily toggles. Settings hold detailed controls. This Home page explains the feature groups and links into Workspace so you can hide what you do not need without deleting saved preferences.

Layout

Align, measure, and mark the page while you design.

⊟

Rulers

What: Pixel rulers along the viewport edges.

Why: Quick distance checks without leaving the page.

How: Toggle from the popup, then drag origin if needed.

⊥

Guides

What: Straight or free-draw alignment lines.

Why: Keep columns, baselines, and edges consistent.

How: Enable Guides, place lines, lock when settled.

⊞

Grids

What: Column and baseline overlays.

Why: Validate rhythm against a layout system.

How: Choose columns/gutters in popup controls.

⟷

Measure

What: Point or rectangle measurements.

Why: Capture exact sizes between elements.

How: Turn on Measure and click the page surface.

⊡

Drawing

What: Temporary annotation marks on the page.

Why: Leave visual notes during review passes.

How: Pick a drawing tool and sketch overlays.

Inspect

Understand structure, breakpoints, and wireframe intent.

⊙

Inspect

What: Hover inspection for box model and type.

Why: See spacing and typography without DevTools noise.

How: Enable Inspect and move across elements.

⊡

Responsive

What: Device-width preview helpers.

Why: Check layouts at common breakpoints quickly.

How: Toggle Responsive and pick a device width.

☶

Wireframe

What: Structural wireframe rendering of the page.

Why: Review hierarchy without visual decoration.

How: Enable Wireframe for a stripped composition view.

Color & media

Sample colors, build palettes, and pull page assets.

✎

Color Picker

What: Eyedropper and color history for the page.

Why: Match brand colors from live UI.

How: Enable Color Picker and sample the screen.

◨

Palette

What: Generated palette from page colors.

Why: Turn a live site into a reusable swatch set.

How: Run Palette and copy the groups you need.

⧉

Media

What: Inventory of images, video, SVG, and more.

Why: Audit and download assets from the current page.

How: Open Media Manager and scan the tab.

Quality

Catch accessibility, consistency, and performance issues early.

≛

Accessibility

What: Issue scan with severity filters.

Why: Surface barriers before shipping.

How: Enable Accessibility and review highlighted findings.

Ӫ

Design Check

What: Consistency checks for type, color, spacing.

Why: Spot drift across a page system.

How: Run Design Check and triage highlighted mismatches.

≉

Page Speed

What: Local performance diagnostics for the tab.

Why: Find heavy assets and slow paths while designing.

How: Open Page Speed and run against the current page.

ↁ

Design System

What: Extracted tokens and UI patterns from the page.

Why: Document what the live UI is actually using.

How: Scan with Design System and export what you need.

Utilities

Everyday helpers for cache, timing, and page behavior.

⦻

Cache Cleaner

What: Selective browsing-data cleanup.

Why: Clear stale state while testing changes.

How: Choose data types and run a clean pass.

⏱

Page Load Timer

What: Visible load timing for the current navigation.

Why: Compare perceived speed after edits.

How: Enable the timer and reload the page.

⊘

Page Controls

What: Temporary page behavior controls while testing.

Why: Isolate issues without changing source code.

How: Toggle the controls you need for the session.

WordPress

Admin-aware tools for builders working on WordPress sites.

W

WP Tools

What: A floating panel for admin bar control, smart edit, role switcher, private theme preview, detector, backlog, notices, and more.

Why: Keep WordPress builder chores in one place instead of hunting admin screens.

How: Enable WP Tools on a WordPress tab. Theme switcher and role switcher use the bundled Aligner WP Tools Helper when needed — install requires plugin upload permission.

Daily control

Hide what you do not need. Keep what you use.

Workspace visibility only changes what appears in the popup. Your saved feature settings stay intact, so turning a group back on restores the previous controls immediately.
