✨ Feature Highlights

Upload & Replace: Easily replace any image with your own uploads or URL images. Supports both IMG tags and background-image CSS.

Image Detection: Automatically detects image elements and shows amber outline instead of blue. Image info displays source URL and dimensions.

Find All Images: Scans the entire page for all images and displays them in a clickable list with thumbnails.

Advanced Manipulation: Crop, flip, remove background, apply filters (grayscale, blur, brightness, contrast, sepia), and shape clipping (circle, triangle, hexagon, star, etc.).

Copy & Reset: Copy CSS only or HTML+CSS. Reset inline styles to defaults. Reset Tailwind classes to original state.

💡 Pro Tips:

  • Press Escape to close the inspector panel
  • Images show amber outline instead of blue when selected
  • Use "Find Images" to quickly locate all images on complex pages
  • Upload supports all image formats: JPG, PNG, GIF, WebP, SVG
  • Background removal uses CSS tricks - may not work perfectly on all images
  • Crop values example: "10px 10px 10px 10px" (top right bottom left)
  • Combine filters for creative effects (blur + grayscale + sepia)
  • Try shape clips on profile images for unique designs
