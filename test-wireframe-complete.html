This page tests all aspects of the Wireframe Generator feature. Use the Aligner extension to generate wireframes with different settings and verify all text modes work correctly.

✓ Test Instructions:

2. Click the "Wireframe" button (☶ icon)
3. Try different settings:
   • Text Mode: Bars → Lines → Keep Real Text
   • Density: Low → Medium → High
   • Border Style: Thin → Medium → Thick
4. Generate wireframe and verify text is preserved with "Keep Real Text" mode
5. Export to HTML and verify the exported file works correctly
